Customer Behavior & Sales Analytics Project

This project showcases an end-to-end data pipelines engineering, SQL relational schema architecture, exploratory data analysis, and advanced fullstack visual dashboarding.

Technical Stack Summary

Py

Python Data Engineering

Created random custom mock dataset mimicking sales seasonality, age segments preferences, and coupon usage. Applied **Pandas** for ingestion, cleansing, logic mapping, and schema formatting.

SQL

Relational Database

Engineered a structured relational schema in **SQLite** containing constraint checks, foreign keys, cascade deletes, and column indexes. Queries match PostgreSQL syntax, demonstrating business logic modeling.

EDA

Exploratory Data Analysis

Used **Matplotlib** and **Seaborn** to build and export standard visual plots of demographics distributions, rating spreads, monthly trends, and product performance.

Web

Dynamic Dashboard

Programmed a frontend console featuring HTML5 grid grids, CSS glassmorphism, **Chart.js** canvases, and a customized **live SQL console** utilizing a lightweight Python Flask backend.

Core Business Insights Discovered

  • Loyalty Subscription Impact: Subscribers exhibit a 10-15% increase in Average Order Value (AOV) and order sizes compared to general clients. They also maintain rating loyalty (higher satisfaction scores on average).
  • Category Profitability: Electronics generates the highest gross revenue despite lower purchase counts, whereas Clothing and Books generate high volume transaction counts but smaller individual basket margins.
  • Q4 Seasonal Spikes: There is an observable 2.5x spike in transaction volumes during November and December, mirroring global retail holiday trends.
